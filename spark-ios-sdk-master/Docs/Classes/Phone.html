    Note

    This function is expected to run on main thread.

    Declaration

    Swift

    public func dial(address: String, option: MediaOption, completionHandler: (Bool) -> Void) -> Call

    Parameters

    address

    Intended recipient address. Supported URIs: Spark URI (e.g. spark:shenning@cisco.com), SIP / SIPS URI (e.g. sip:1234@care.acme.com), Tropo URI (e.g. tropo:999123456). Supported shorthand: Email address (e.g. shenning@cisco.com), App username (e.g. jp)
    option

    Media option for call: audio-only, audio+video etc. If it contains video, need to specify render view for video.
    completionHandler

    A closure to be executed once the action is completed. True means success, and False means failure.

    Return Value

    Call object
